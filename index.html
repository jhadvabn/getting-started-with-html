Introduction to Programming

In this introductory course we will learn how to program using HTML.
HTML stands for HyperText Markup Language. HTML uses tags and attributes to define how the content is displayed.
Some of the tags that we learned are:

  • b = bold
  • br = break
  • p = paragraph
  • img = image
  • a = anchor

    • FYI - the above needs the less than and greater than signs around them but I'm not sure how to display them as text.

    Inline vs. Block Tags
    Tags are either inlineor block. Block tags such a "P" for paragraph have width and height
    while inline tags do not.

    Attributes
    Tags can also include an attribute which give the developer more control over what a webpage can do.
    For example the anchor tag (a) can include an 'href' attribute which references a URL.

    i.e. CNN
    'img src' is another attribute for referencing an image within a webpage.
